# Discord Clone Projesi

Bu proje, Discord benzeri gerçek zamanlı bir sohbet ve iletişim platformudur.

## 🛠 Kullanılan Teknolojiler

### Frontend
- React.js
- TypeScript
- Tailwind CSS
- Socket.io Client
- WebRTC

### Backend
- Node.js
- Express.js
- Socket.io
- MongoDB
- Node-Cache (Önbellek için)
